# AI-Assisted Data Structures Practice

Este repositório contém a resolução de desafios de programação no estilo code interview, utilizando apoio de ferramentas de Inteligência Artificial para auxiliar no desenvolvimento, revisão e entendimento das soluções.

---

## Objetivo

Praticar estruturas de dados fundamentais e desenvolver habilidades de resolução de problemas com suporte de IA.

---

## Ferramenta de IA Utilizada

- **Nome:** Gemini
- **Acesso:** Via interface Web (navegador), atuando como um assistente conversacional paralelo ao ambiente de desenvolvimento.
- **Por que foi escolhida para esta atividade:** A escolha se deu pela capacidade da IA de explicar algoritmos complexos de forma simples, linha por linha. Ela se mostrou ideal para ajudar a prever onde o código pode falhar e para validar se a lógica pensada está correta. Isso é essencial para entendermos o funcionamento do programa a fundo e nos ajuda a pensar nas melhores formas de testá-lo para garantir a qualidade.
- **Como foi usada:** A interação ocorreu de forma orgânica via chat, dispensando tutoriais ou vídeos. A codificação e os testes locais foram iniciados no VS Code, com a execução e validação final sendo feitas diretamente na plataforma do LeetCode. A IA foi mantida em uma aba lado a lado no navegador. Essa separação permitiu usar a IA como uma "revisora de código" externa, mantendo o VS Code e o LeetCode focados puramente na escrita e execução dos scripts.
- **Como a equipe utilizou:**
  - Explicar detalhadamente a lógica estrutural e a complexidade (Tempo e Espaço) das soluções.
  - Revisar algoritmos para identificar gargalos e riscos potenciais.
  - Mapear cenários críticos de teste e sugerir abordagens de validação (como testes de limite e comportamento com dados nulos/duplicados).
  - Gerar documentação técnica limpa e comentar o código em blocos lógicos, facilitando a manutenção e a rastreabilidade da qualidade.
